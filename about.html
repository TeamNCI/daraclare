Hi, I'm Dara Clare, designer & developer

Designer turned developer

Starting as a designer in 2005, I've worked in various SMEs as the sole designer for print and web, as well as working for large corporations such as Bank of Ireland and Deloitte. With a healthy obsession with typgraphy, I received a first class honours MA from DIT, which included a 20,000 word thesis on Responsive Web Typography

Working with code (namely HTML and CSS) since 2005, the decision to retrain as a developer was an obvious step for me. I have always enjoyed problem solving and debugging code. Below are a few topics which remain the most interesting aspects of web development and coding for me:

  • Accessibility: creating code which is accessible by screen readers and other devices to create an all inclusive, user-friendly experience
  • Responsive design: from responsive logos to typography
  • Search engine optimisation: a necessary evil!
  • Prototyping: revealing how a design will look and interact with its users
Graphic Design Dublin

Graphic Design Dublin is a meetup group I started in 2012 for web and graphic designers in Dublin. It was created with a view to providing a networking platform for Dublin based designers and developers and now has over 430 members.

HTML & CSS Courses

In partnership with CodingGrace and DPP Skillnet, we run HTML & CSS Courses aimed for designers of all levels. If you interesting in joining us, please visit CodingGrace.com for more information and to find upcoming events.

Typography Courses

In partnership with DPP Skillnet, we will soon be running a typography course for designers. Topics will include laying out type in a grid, implementing Typekit, choosing and combining typefaces and more. Further informaton to follow very soon!
